# Cocos2d / Cocos Chat Export Archive

Last updated: 2026-07-07

This folder is for raw exports from the older Cocos/Cocos2d prototype chat and reviewed extracts from those exports.

## Authority

Cocos/Cocos2d chat exports are archive/reference material. Active development is in Unity.

Use this order when reconciling decisions:

1. Current user instruction in the active thread.
2. `CURRENT_PROTOTYPE_STATE.md`.
3. `Docs/LOCKED_DECISIONS.md`.
4. `Docs/10_VERIFIED_PROJECT_STATE.md`.
5. `Docs/09_OPEN_DECISIONS.md`.
6. Reviewed Cocos decision extracts in this folder.
7. Raw Cocos chat exports in this folder.
8. Older Cocos project files.

Do not treat Cocos implementation details as active Unity requirements unless a current locked doc or user instruction adopts them.

## Recommended File Names

- Raw export: `YYYY-MM-DD-cocos-chat-export.md`
- Reviewed extract: `YYYY-MM-DD-cocos-chat-decision-extract.md`

## Import Workflow

1. Place the raw Cocos/Cocos2d chat export in this folder.
2. Extract decisions, useful rationale, and old implementation context.
3. Mark anything superseded by Unity/current docs.
4. Reconcile only confirmed/current items into locked/open/verified docs.
