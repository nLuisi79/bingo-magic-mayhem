# ChatGPT Project Export Archive

Last updated: 2026-07-07

This folder is for raw ChatGPT project/chat exports and reviewed extracts.

## Authority

Raw chat exports are searchable reference material, not source-of-truth by themselves.

Use this order when reconciling decisions:

1. Current user instruction in the active thread.
2. `CURRENT_PROTOTYPE_STATE.md`.
3. `Docs/LOCKED_DECISIONS.md`.
4. `Docs/10_VERIFIED_PROJECT_STATE.md`.
5. `Docs/09_OPEN_DECISIONS.md`.
6. Reviewed decision extracts in this folder.
7. Raw chat exports in this folder.

## Recommended File Names

- Raw export: `YYYY-MM-DD-chatgpt-project-export.md`
- Reviewed extract: `YYYY-MM-DD-chatgpt-project-decision-extract.md`

## Import Workflow

1. Place the raw ChatGPT export in this folder.
2. Search/extract decisions from the raw export.
3. Write a reviewed decision extract using `DECISION_EXTRACT_TEMPLATE.md`.
4. Reconcile only confirmed/current items into locked/open/verified docs.
5. Mark superseded or conflicting ideas clearly instead of silently adopting them.

## What To Extract

- Confirmed decisions
- Strong preferences
- Open decisions
- Superseded ideas
- Contradictions with locked docs
- Implementation details already agreed
- Future risks
